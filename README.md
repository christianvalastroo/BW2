# Airbnb Clone – Frontend Project 🏡

Questo progetto è un **clone semplificato della piattaforma Airbnb** sviluppato come esercitazione di **HTML, CSS, Bootstrap e JavaScript**.

L'obiettivo è replicare alcune delle principali interfacce del sito Airbnb per allenarsi su:

- layout responsive
- componenti Bootstrap
- organizzazione del codice frontend
- navigazione tra pagine
- gestione delle card e delle sezioni

---

## 📁 Struttura del progetto

Il progetto è composto da **tre pagine principali**.

### Home page
File: `index.html`

Contiene:

- navbar responsive
- sezione hero con immagine e call to action
- barra di ricerca stile Airbnb
- sezione destinazioni
- footer

Questa pagina rappresenta la **landing page del sito**.

---

### Catalogo delle case
File: `indexAirbnb2.html`

Questa pagina mostra:

- titolo della città selezionata
- filtri delle case
- griglia di card con gli appartamenti disponibili

Ogni card contiene:

- immagine
- rating
- tipo di alloggio
- località
- nome della casa

Le card sono disposte con **Bootstrap Grid**.

---

### Pagina dettaglio
File: `dettaglio.html`

Questa pagina rappresenta la **scheda di una casa specifica**.

Contiene:

- gallery immagini
- informazioni sull'alloggio
- descrizione host
- servizi della casa
- form per inserire recensioni
- sezione commenti utenti
- card prenotazione con date e ospiti

---

## 🛠 Tecnologie utilizzate

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript
